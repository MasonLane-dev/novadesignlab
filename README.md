# NovaWeb Studios — Portfolio Website

Your professional web design business portfolio.

## Quick Start

Open `index.html` in a browser to preview locally.

## Live URL

**https://novadesignlab.netlify.app**

## Deploy to Netlify (Free)

1. Go to https://netlify.com and sign up (free)
2. Drag-and-drop the `portfolio-website` folder onto Netlify
3. Your site is live at `https://random-name.netlify.app`
4. Go to Site Settings → Change site name

## Custom Domain (Optional)

Buy a custom domain from Cloudflare/Namecheap (~$11/yr)
then set up custom domain in Netlify settings.
